Разработка инженерного программного обеспечения

Участники проекта "Разработка инженерного программного обеспечения" используют следующий стек для разработки:

  • API nanoCAD, API Компас-3D
  • С/C++, Qt, MFC
  • C#, .NET, WPF
  • Python, FastAPI, SQLite
  • JS, React
  • Docker, DevContainers
  • OpenGL, Babylon.js
